# ATM System

A complete ATM System developed in C# using .NET 8, Entity Framework Core, and SQL Server.

## Features
- User Registration
- Secure Login
- Account Lock After 3 Failed Login Attempts
- Deposit and Withdraw
- Money Transfer
- Transaction History
- Check Balance
- Edit Profile
- Delete Account
- Admin Panel
- Export Transactions to CSV
- Search Account

## Technologies Used
- C#
- .NET 8
- Entity Framework Core
- SQL Server
- Git
- GitHub
